FAQ

I have too old composer version.

You should contact your hosting to upgrade the composer version.
If there's no such possibility - you can enable the Download composer.phar to store root checkbox. It will put the latest composer version to your store.

I get Authentication required (repo.magento.com): prompt

You can enter your Magento API credentials from Magento Site. Or go to composer.json and remove following code at bottom during installation, then don't forget to put it back: "0": { "type": "composer", "url": "https://repo.magento.com/" },

I have an Argento installed already. Just want to upgrade.

Please, go to your store root and run following commands:
composer update swissup/argento-m2 --with-all-dependencies

php bin/magento setup:upgrade

rm -rf pub/static/_requirejs var/view_preprocessed pub/static/frontend/ pub/static/adminhtml/ generated/code/

php bin/magento setup:static-content:deploy en_US -f
